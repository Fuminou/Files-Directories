How to compile my program:
Run the following commands
Step 1- gcc --std=gnu99 -o movies_by_year main.c
Step 2- ./movies_by_year movies_sample_1.csv

Thats it :D

This program will:
1.Read directory entries
2.Finds a file in the current directory based on user specified criteria
3.Reads and processes the data in the chosen file
4.Creates a directory
5.Creates new files in the newly created directory and writes processed data to these files
